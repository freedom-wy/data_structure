## 数据结构与算法
### 冒泡排序
```python
'''
用索引值去比较，低位索引和后一位索引比较，然后进行交换
'''
alist = [77,1,22,55,9,0,6,-10,100]

n = len(alist)

for j in range(0,n):
    # 最后一位无需选取,下面的判断语句会将索引值+1
    for i in range(0,n-1-j):
        if alist[i] > alist[i+1]:
            alist[i],alist[i+1] = alist[i+1],alist[i]
print(alist)
```
