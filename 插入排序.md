## 数据结构与算法
### 插入排序 
```python
alist = [9,2,-1,6,10]

n = len(alist)
# 取出一个值，判断下一个值和前一个值的大小
for i in range(1,n):
    while i > 0:
        if alist[i] < alist[i-1]:
            alist[i-1],alist[i] = alist[i],alist[i-1]
            i = i-1
        else:
            break

print(alist)
```
